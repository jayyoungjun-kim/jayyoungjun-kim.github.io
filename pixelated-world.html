Pixelated World
픽셀화된 세계는 실시간으로 색상을 감지하고 이미지를 픽셀 형태로 변환하는 프로젝트입니다. 픽셀의 크기는 주변 밝기에 따라 다섯 가지 크기로 변화하며, 이를 통해 우리가 바라보는 세상을 극단적으로 단순화하여 새로운 시각적 경험과 인식의 전환을 유도하는 것이 목표였습니다. 이 시스템의 기본 원리는 환경의 밝기에 따라 픽셀 크기가 조정되는 방식입니다. 주변이 밝아지면 픽셀이 작아지고, 어두워지면 커지는 구조로, 빛과 어둠을 단순화된 형태로 체험할 수 있도록 설계되었습니다. 또한, 매 순간 무작위로 색상이 조합되며, 주변 환경이 단순하면서도 동시에 다채롭게 표현되는 효과를 제공합니다. 이를 통해 사용자는 익숙한 세상을 새로운 방식으로 바라볼 수 있는 시각적 경험을 하게 됩니다.
The Pixelated World is a project that detects colors in real-time and transforms images into a pixelated format. The size of the pixels changes into five different scales depending on the surrounding brightness. The goal of this project was to simplify the way we perceive the world to an extreme level, providing a new visual experience and shifting our perspective. The core principle of this system is adjusting pixel size according to ambient brightness. When the surroundings become brighter, the pixels shrink, and when it gets darker, the pixels enlarge. This structure allows users to experience light and darkness in a simplified form. Additionally, colors are randomly combined every moment, creating an effect where the environment appears both simplified and vibrant at the same time. Through this, users can engage in a visual experience that reinterprets the familiar world in a new way.
