DESIGENE
DESIGENE 서체는 서체의 가변성과 유연성을 탐구하는 프로젝트입니다. 기존의 폰트는 완성된 형태로 제작되어 배포되며, 사용자는 이를 그대로 다운로드하고 설치해야 하는 고정된 구조를 가지고 있습니다. 즉, 사용자가 직접 수정하지 않는 한, 폰트 자체가 변화할 수 없는 정적인 형태입니다. DESIGENE 서체는 이러한 한계를 넘어, DNA의 형태를 모티브로 한 유전자 변형 서체 개념을 제안합니다. 사용자가 DESIGENE 폰트를 다운로드하고 설치하면, 글자와 문자들이 위에서 아래로 무작위로 배열되며, 다양한 크기의 입자들로 실시간 자동 생성됩니다. 이를 통해 폰트가 고정된 형태를 벗어나, 계속해서 변화하는 유기적인 타이포그래피 시스템으로 작동하게 됩니다.
The DESIGENE typeface is a project that explores the variability and flexibility of typefaces. Traditional fonts are designed in a fixed form, completed, and distributed for users to download and install as they are. In other words, fonts remain static and unchangeable unless manually modified by the user. DESIGENE challenges this limitation by introducing a genetically modified typeface concept inspired by the structure of DNA. When users download and install the DESIGENE font, letters and characters are randomly arranged from top to bottom and are automatically generated in real time using particles of varying sizes. This allows the typeface to break away from a fixed structure and function as an ever-evolving, organic typographic system.

폰트 적용
어떠한 폰트도 본연의 형태가 자유롭게 변화할 수 있다
DESIGENE 서체는 헬베티카 볼드와 윤고딕 볼드를 기반으로 Nodebox↗를 사용하여 프로그래밍되었습니다. 이 시스템의 형태와 패턴을 활용하면, 폰트는 어떤 형태로든 자유롭게 변화할 수 있습니다.
The DESIGENE typeface is programmed using Nodebox↗, based on Helvetica Bold and YoonGotic Bold. With the shape and pattern of this system, the font can freely transform into any form.
