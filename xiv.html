IBM XiV E-brochure
IBM XiV E-brochure is designed and developed based on a tablet application to efficiently promote IBM XiV System, especially GEN3, which is a disk storage server. Contents are divided by the grid and color system to help people get vast information effectively.
Grid
The grid system is intended to systematically categorize each content for getting complex information effectively. the reason why I used a 3-stage grid system is that title, sub-title 1 and sub-title 2 are the basic structure of the content.
Location, Font Size
The arrow which is an orange color an intuitive indicator to inform you of where the content is belonged to now. In the same sense, the 3-stage font size follows the content hierarchy.
Content
If it is not enough to put each content to each category based on a 3-stage grid system such as title, sub-title 1 or sub-title 2, the area of content can be flexibly expanded to other areas as various forms by being divided.
